# **App Name**: Ginim's Digital Domain

## Core Features:

- Navbar: Sticky top navbar with transparent to solid transition on scroll. Includes sections for Home, About, Skills, Projects, Achievements, and Contact. Features animated underline on hover and LinkedIn/Email icons.
- Hero Section: Bold, futuristic hero section with 'Hi, I'm Ginim Narang' headline, 'Software Developer | Tech Enthusiast | Problem Solver' subhead, and an animated roles ticker. Includes Download Resume (primary) and View Projects (outline neon) CTA buttons. Features a futuristic abstract 3D tech sphere or glowing gradient blob or geometric rotating wireframe with slow floating elements and hover magnetic effect on buttons.
- About Me: Short, clean card-based design including name, location (New Delhi, India), education (BCA Student at BVIMR 2024–2027), CGPA (9.25), and a short paragraph describing passion for coding, projects, system design, and hackathons. Features glowing badges for Hackathon Winner, Active Coding Club Member, Internship Completed, and Mentor for Junior Projects.
- Tech Skills: A grid of glowing tech cards with icons. Skill categories include Languages (C, C++, Java, Python, JavaScript), Frontend (HTML, CSS, JavaScript), Backend (API Integration, DBMS), Tools (VS Code, Git, GitHub), and Other (System Design, OOP, Financial Tools).  Includes tilt-on-hover animations, neon borders, and skill-level bars or animated dots.
- Projects Section: Showcase 6 high-quality project cards with hover expand animations. Projects include Attendance Monitoring System, Sales Management App, Dynamic Mini Projects, and Trading & Financial Tools.  Each project card includes a thumbnail, tech stack icons, short description, and a 'View More' CTA.
- Hackathons & Achievements: Stylish timeline with neon connectors, showcasing participation in national & regional hackathons, college-level competitions won, organization of technical fests & coding competitions, mentorship of juniors, and highlighting strong collaboration & problem-solving achievements.
- Contact Section: Clean minimal contact section with email (ginim654@gmail.com) and LinkedIn (linkedin.com/in/ginim-narang-631924283) information.  Features a glowing contact form with smooth form success animation, floating icons, and a subtle moving gradient lines + particles background.
- Footer: Techy dark footer with glowing border top. Includes Quick links, Social icons, copyright notice ('© 2025'), and a 'Made with ❤️ + Code' statement.
- Extra Features: Includes cursor glow trail, 3D tilt hero image, floating hologram icons, scroll-triggered fade/slide, preloader with rotating code brackets, dynamic theme color changer, animated number counters for achievements, and a downloadable resume button that uses your PDF.

## Style Guidelines:

- Primary color: Electric blue (#0A84FF) to represent technology and innovation.
- Background color: Light blue (#EAF1FF) for a clean, modern aesthetic.
- Accent color: Deep navy (#0F172A) to add depth and contrast for key UI elements.
- Headline font: 'Poppins' (sans-serif) for a bold, contemporary feel.
- Body font: 'Inter' (sans-serif) for clean readability and a modern look.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use minimalist line icons for skills and technologies with a neon glow effect on hover.
- Implement a clean grid layout with generous spacing, glassmorphic cards, and floating UI elements to achieve a futuristic look.
- Incorporate smooth parallax scrolling, subtle micro-interactions, and animated reveals to enhance user experience and visual appeal.
